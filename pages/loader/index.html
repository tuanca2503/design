Demo PageLoader

Mở Console (F12) để xem log. Nhấn Ctrl+F5 để tải lại.

Thanh load nằm trên đỉnh trang. Nó đợi: font, ảnh banner, 2 API giả lập, cùng 2 mốc mặc định (DOMContentLoaded + load).

Mốc data (3.5s) cố tình chậm hơn window.load, nên thanh sẽ giữ ở mức cao rồi mới chốt 100% khi nó xong — đúng như đã bàn.
